# Hussein Fans Counter

موقع هدية بعنوان: **عدد محبين حسين حيدر العطار**.

## الميزات
- 6 قلوب ملونة تتغير بترتيب عشوائي بسرعة.
- عداد فعلي يزيد عند ضغط الزر.
- واجهة عربية لطيفة ومتوافقة مع الجوال.

## النشر على Vercel
المشروع جاهز للنشر مباشرة.

لجعل العداد **فعلي ومشترك** بين كل الزوار على الإنترنت، أضف متغيرات البيئة التالية في Vercel:

- `UPSTASH_REDIS_REST_URL`
- `UPSTASH_REDIS_REST_TOKEN`
- `COUNTER_KEY` (اختياري، الافتراضي: `hussein_fans_count`)

> مهم: بدون Redis لن يعمل العداد في هذه النسخة، وهذا مقصود لضمان أن العداد يكون فعليًا ومشتركًا.

## اختبار محلي
بعد تثبيت Node.js و Vercel CLI:

```bash
npm i -g vercel
vercel dev
```

ثم افتح:

`http://localhost:3000`
